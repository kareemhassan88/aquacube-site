# AquaCube Site

## Project Overview
AquaCube is a water treatment technology company. This is the marketing website built with React 18 + Vite.

## Tech Stack
- React 18.2 with Vite 5.0
- Inline CSS styles (no external CSS framework)
- Deployed on Netlify

## Skills
- **UI/UX Pro Max** (`ui-ux-pro-max`): Design intelligence skill for UI/UX work. Automatically activates for design, build, and styling tasks. Use it for color palettes, typography, component design, accessibility, and layout decisions.

## Development
- `npm run dev` - Start dev server
- `npm run build` - Production build (outputs to `dist/`)

## Design Guidelines
- Primary brand colors: Aqua (#1FB4BA), Navy (#093D79)
- Bilingual support: English + Arabic (RTL)
- Mobile-first responsive design
